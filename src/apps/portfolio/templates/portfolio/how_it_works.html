Как работает сервис

Короткая справка по каждому элементу интерфейса.

MOEX + Марковиц + Monte Carlo

Шаг 1. Ввод портфеля

  • Тикер — код бумаги (например, SBER, LKOH).
  • Вес, % — текущая доля бумаги в вашем портфеле.
  • Средняя цена — вручную нужна только в демо-режиме без MOEX.
  • Загрузить из MOEX — подтягивает реальные исторические цены по тикерам.
  • Ручной -> JSON — переводит форму в JSON для отладки и ручной правки.

Шаг 2. Параметры расчета

  • Безрисковая ставка — используется в коэффициенте Sharpe (по умолчанию 12.5% = 0.125).
  • Профиль цели — выбор целевого портфеля: консервативный/сбалансированный/агрессивный.
  • Портфелей Monte Carlo — сколько случайных портфелей генерировать.
  • Точек эффективной границы — детализация кривой Марковица.
  • Порог ребалансировки — минимальное отклонение веса для действия buy/sell.
  • Горизонт и сценарии прогноза — параметры моделирования итоговой стоимости.

Что означают блоки результатов

  • Ключевые показатели — доходность, риск, Sharpe, диверсификация.
  • Оценка портфеля — интегральный скоринг 0..10 (эвристика для быстрого сравнения).
  • Сводка риска — сравнение текущего и целевого портфеля (VaR, CVaR, просадка).
  • Ребалансировка — таблица действий: что увеличить/сократить и приоритет.

Графики

  • Эффективная граница — баланс риска и доходности по Марковицу.
  • Monte Carlo портфели — облако случайных комбинаций с выделением лучших точек.
  • Матрица корреляций — взаимосвязь доходностей активов.
  • Распределение итоговой стоимости — прогноз диапазона результата портфеля.
